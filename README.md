## Quick start

```commandline
git clone https://github.com/evgcrvsed/Trader.git
```

```commandline
pip install -r requirements.txt
```

Создай файл app/.env
```commandline
STEAM_LOGIN='login'
STEAM_PASSWORD='passw'
STEAM_SHARED_SECRET='secret'
STEAM_IDENTITY_SECRET='secret'
STEAM_STEAM_ID=123
```
